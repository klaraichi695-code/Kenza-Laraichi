Système de Gestion Bancaire – Java

Application Java développée dans le cadre d’un projet académique afin de mettre en pratique les concepts de base de la Programmation Orientée Objet.

Fonctionnalités:

-Gestion des clients
-Comptes bancaires (courant, épargne, entreprise)
-Dépôt, retrait et historique des transactions

Bonus : notifications (simulation)

Concepts utilisés:

-Héritage
-Polymorphisme
-Encapsulation
-Abstraction
